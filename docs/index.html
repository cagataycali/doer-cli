01Install

One line. Python 3.10+. Ollama running locally (or `OLLAMA_HOST` set).

# from PyPI
pip install doer

# or grab a prebuilt binary (no Python needed)
curl -sSL https://github.com/cagataycali/doer/releases/latest/download/doer-$(uname -s | tr A-Z a-z)-$(uname -m) \
  -o /usr/local/bin/doer && chmod +x /usr/local/bin/doer

02Run

Pipe anything in. Pipe anything out. Chain it like `grep`.

doer "find files larger than 100MB"

cat error.log  | doer "what broke"
git log -20    | doer "write release notes"
echo '{"a":1}' | doer "to yaml"
curl -s api.io | doer "summarize"  | tee out.md

03Context

Every call, the agent reads your shell like a person reads a room. No database. No config. The filesystem is the memory.

SOUL.md

Drop a SOUL.md in cwd. It defines who doer is inside this project.

AGENTS.md

Drop an AGENTS.md in cwd. Rules, conventions, don't-dos.

~/.doer_history

Last N Q/A pairs in zsh-compatible format. Env: DOER_HISTORY.

~/.bash_history + ~/.zsh_history

Merged chronologically. Last N commands. Env: DOER_SHELL_HISTORY.

./tools/*.py

Any @tool function. Hot-reloaded by Strands on every call.

Own source

Full doer/__init__.py is in the system prompt. Full self-awareness.

05Extend

Drop a Python file in ./tools/. Next call, it's live.

# ./tools/weather.py
from strands import tool
import urllib.request

@tool
def weather(city: str) -> str:
    """Weather for a city."""
    return urllib.request.urlopen(f"https://wttr.in/{city}?format=3").read().decode()

→ doer "weather istanbul?" — no restart needed.
